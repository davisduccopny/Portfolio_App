Announcing the Python Developer Tooling Handbook

Coming soon: a free ebook about Python developer tooling!

Feb 10, 2024

Introduction

I've been a professional Python developer for over a decade. Like many Python developers, I've faced the many challenges of Python packaging and dependency management. I've also come to love helping developers find the right tools to make their work easier and more productive.

Why I Wrote This Handbook

I'm excited to announce that I'm writing the Python Developer Tooling Handbook, a free ebook on Python developer tooling.

The Pain of Tooling

Many new developers struggle with setting up a proper Python development environment. Between virtual environments, dependency management, and code formatting, there's a lot to juggle.

The Vision

This handbook aims to become the go-to resource for understanding and setting up professional-level tooling in Python projects.

What Will Be Covered

This is a brief overview of what you'll find inside the handbook:

Development Environment

Virtual Environments

Learn how to use tools like venv, virtualenv, and pipenv to manage isolated environments.

Editors and IDEs

Tips on configuring VSCode, PyCharm, and other editors to suit Python development.

Dependency Management

Using requirements.txt

Classic way of pinning dependencies in your Python project.

Using pyproject.toml

Modern tools like Poetry and PDM prefer this file for centralized configuration.

Pros and Cons

We'll weigh the benefits of adopting modern tools versus sticking with traditional workflows.

Stay Updated

The book is currently a work in progress. If you'd like to be notified once it's released, sign up for the mailing list.
